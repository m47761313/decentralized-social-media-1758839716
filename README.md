# Decentralized Social Media Platform

A censorship-resistant social platform with user-owned data and content monetization using smart contracts on the Stacks blockchain.

## Overview

This project implements a decentralized social media system where users maintain complete ownership of their data and content while enabling direct monetization through blockchain technology. The platform eliminates centralized control and censorship concerns.

## Features

- **Censorship Resistance**: Immutable content storage on blockchain
- **User Data Ownership**: Complete control over personal data and content
- **Content Monetization**: Direct revenue sharing with creators
- **Decentralized Identity**: Self-sovereign identity management
- **Community Governance**: User-driven platform decisions
- **Privacy Controls**: Granular content visibility settings

## Smart Contracts

### social-network
The core contract that handles:
- Post and interaction storage with user control
- Data ownership and access permissions
- Revenue sharing mechanisms for content creators
- Community governance and moderation tools

## Technical Architecture

Built with Clarinet and Clarity smart contracts on the Stacks blockchain, ensuring:
- Immutable content storage
- Transparent monetization
- User-controlled data access
- Decentralized content distribution

## Key Benefits

- **For Users**: Complete data ownership and privacy control
- **For Creators**: Direct monetization without intermediaries
- **For Communities**: Self-governed content moderation
- **For Developers**: Open protocol for social applications

## Getting Started

1. Install Clarinet: `npm install -g @hirosystems/clarinet`
2. Clone the repository
3. Run `clarinet check` to validate contracts
4. Deploy to testnet for testing
